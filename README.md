# Minhas Finanças

Aplicativo de controle financeiro pessoal desenvolvido em Android nativo com Jetpack Compose.

## Funcionalidades

- **Controle de Contas:** Gerencie múltiplas contas bancárias, carteiras e investimentos.
- **Categorização:** Organize suas receitas e despesas por categorias personalizáveis.
- **Lançamentos:** Registre todas as suas transações financeiras de forma simples e rápida.
- **Autenticação:** Acesso seguro com email/senha e suporte a biometria.
- **Interface Moderna:** Desenvolvido inteiramente com Material Design 3 e Jetpack Compose.

## Tecnologias Utilizadas

- **Linguagem:** Kotlin
- **UI:** Jetpack Compose
- **Injeção de Dependências:** Hilt
- **Banco de Dados:** Room
- **Arquitetura:** MVVM (Model-View-ViewModel)
- **Navegação:** Compose Navigation
- **Persistência de Preferências:** DataStore

## Configuração do Projeto

1. Clone o repositório.
2. Abra o projeto no Android Studio (versão Ladybug ou superior recomendada).
3. Sincronize o Gradle.
4. Execute o aplicativo em um emulador ou dispositivo físico com Android 8.0 (API 26) ou superior.
